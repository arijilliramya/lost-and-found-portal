Lost and Found Campus Portal:
Overview
The Lost and Found Campus Portal is a simple web-based project that allows students and staff to report and track lost or found items on campus. Users can submit items they have lost or found, and search for matching items to help return them quickly.
Features
Report Lost Item: Students can submit details about items they have lost.
Report Found Item: Students can submit details about items they have found.
Search Items: Users can search for lost or found items using keywords.
Technology Used
Frontend: HTML, CSS
